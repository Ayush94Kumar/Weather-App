# Weather App

 weather application that allows users to fetch real-time weather updates for any city around the world.

## Features
- **Real-Time Data:** Get accurate and current weather conditions instantly.
- **Detailed Metrics:** Displays Temperature (°C), "Feels Like" temperature, Wind Speed (km/h), and current weather condition text.
- **Location Awareness:** Shows the specific city name along with its region and country.
- **Error Handling:** Gracefully alerts the user if a city is not found or if the search input is left empty.

## Usage
1. Launch the application in your browser.
2. Type the name of a city (e.g., "London" or "Tokyo") into the input field.
3. Click the **Search** button.
4. View the up-to-date weather statistics displayed in the result card below.
